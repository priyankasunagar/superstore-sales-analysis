# superstore-sales-analysis
Beginner data analysis project using Python and Pandas to analyze Superstore sales data, perform data cleaning, and extract insights like top products, customers, and sales trends.
# 📊 Superstore Sales Analysis

## 📌 Project Overview
This project analyzes a retail Superstore dataset using Python and Pandas.  
The goal is to clean the data and extract useful business insights.

---

## 🛠️ Tools Used
- Python
- Pandas
- Google Colab

---

## 📂 Dataset
- Superstore Sales Dataset (CSV file)

---

## 🔍 Tasks Performed
- Loaded and explored dataset
- Checked and handled missing values
- Removed duplicate data
- Data filtering and selection
- Grouped data using `groupby()`

---

## 📈 Key Analysis
- Total Sales Calculation
- Sales by Region
- Top 5 Products by Sales
- Top 5 Customers by Sales
- Monthly Sales Trend Analysis

---

## 📊 Key Insights
- Identified high-performing regions
- Found top-selling products
- Analyzed customer contribution
- Observed sales trends over time

- Improved data cleaning skills
- Understood groupby operations
